# E-Commerce Customer Segmentation (RFM Analysis)

## 🎯 Project Goal
To transform raw transactional data into actionable marketing segments using Unsupervised Machine Learning. This helps businesses move from "One-size-fits-all" marketing to personalized strategies.

## 🛠 Tech Stack
- **AI/ML:** Scikit-learn (K-Means Clustering)
- **Data Engineering:** Pandas, NumPy
- **Preprocessing:** StandardScaler for feature normalization
- **Tools:** Python, VS Code

## 🧠 Business Logic (RFM)
- **Recency:** Time since last purchase.
- **Frequency:** Total number of purchases.
- **Monetary:** Total revenue generated.

## 📊 Strategic Insights
- **Champions:** Target with early access to new products.
- **At Risk:** Target with "We Miss You" discount codes.
- **Loyal Customers:** Reward with a loyalty program.
